# JavaScript Mini Projects

This repository contains a collection of JavaScript mini projects built from scratch using HTML, CSS, and vanilla JavaScript. The goal of this repository is to strengthen core JavaScript fundamentals through hands-on practice and real-world examples.

## Objectives
- Understand JavaScript basics and ES6 features
- Practice DOM manipulation and event handling
- Implement input validation and user interactions
- Build interactive UI components without frameworks
- Improve problem-solving and debugging skills

## Projects Included

### 1. Counter App
- Increase, decrease, and reset counter value
- Dynamic color change based on value
- Demonstrates DOM manipulation and event handling

## How to Run
1. Clone the repository
2. Open any project folder
3. Open the HTML file in a web browser

## Learning Outcomes
- Strong understanding of JavaScript fundamentals
- Confidence in handling DOM events and user input
- Ability to build interactive web applications without libraries
